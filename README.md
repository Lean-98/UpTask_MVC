## Installation

Recuerden realizar el ```npm install```, para reconstruir los módulos de node e instalar gulp de manera global y composer:
```bas
npm i
npm i -g gulp
composer i
```

Luego ``` gulp``` para ejecutar la aplicación y para el localhost: 
``` 
cd public
php -S localhost:3000 
```
